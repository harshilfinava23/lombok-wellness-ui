# Lombok Wellness UI

A wellness & lifestyle landing page built with pure **HTML & CSS**.
This project focuses on understanding **Responsive Layouts, Flexbox, and Modern UI Design**.

🔗 **Live Demo:** [lombok-wellness-ui.vercel.app](https://lombok-wellness-ui.vercel.app)

## About

The goal was to recreate a modern wellness website UI design as close as possible using:

* Flexbox
* Responsive layouts
* Position properties
* Clean typography
* Section spacing

No JavaScript frameworks. Pure HTML & CSS only.

---

## Tech Used

* HTML5
* CSS3
* Tailwind CSS CDN
* Remix Icons
